# Food Recipe

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Funmicharis/foodapp.git
   
2. Navigate to the project directory::
   ```bash
   cd your-repository
   
3. Install dependencies::
   ```bash
   flutter pub get
